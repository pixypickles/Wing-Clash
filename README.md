# Bird Bowl（1人用 AI対戦版 v2）

`index.html` だけで動作します。GitHub Pagesではリポジトリ直下へ配置してください。

## 操作
- 方向キー：移動／技の方向指定
- C：羽ばたき
- Z：A攻撃
- X：B突進

### A攻撃
- 上＋A：蹴り上げ
- 横または無入力＋A：突き
- 下＋A：叩き落とし

### B突進
- 上＋B：斜め上突進
- 横または無入力＋B：横突進
- 下＋B：斜め下突進

スマホでは左下の円形方向キーと、右下の A・B・翼ボタンを使います。黄色がプレイヤー、青色がAIです。
