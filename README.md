# Bird Bowl Prototype

お椀型ステージで鳥がボールを相手側上部の巣へ叩き込む、ブラウザ向け2人対戦プロトタイプです。ビルド環境は不要で、そのまま GitHub Pages に公開できます。

## 実装済み

- 重力あり、回数無制限の羽ばたき
- 1回の羽ばたきでステージ高のおよそ1/6を上昇
- 横一直線の突進
- 蹴り上げ
- 叩き落とし
- 全攻撃の特大ノックバック
- お椀型ステージ
- 中央へ集まるボール
- 強く短い上昇気流（約4.5〜6.5秒間隔、約0.72秒持続）
- 鳥とボールの両方が上昇気流の影響を受ける
- 左右上部の巣型ゴール
- ローカル2人対戦

## 操作

### プレイヤー1

- 移動: `A` / `D`
- 羽ばたき: `W`
- 横突進: `F`
- 蹴り上げ: `G`
- 叩き落とし: `H`

### プレイヤー2

- 移動: `←` / `→`
- 羽ばたき: `↑`
- 横突進: `J`
- 蹴り上げ: `K`
- 叩き落とし: `L`

## ローカル起動

`index.html` をブラウザで開くだけでも動きます。ローカルサーバーを使う場合:

```bash
python -m http.server 8000
```

その後 `http://localhost:8000` を開きます。

## GitHub Pages で公開

1. このフォルダーの中身を GitHub リポジトリへ追加して push
2. リポジトリの **Settings → Pages** を開く
3. **Deploy from a branch** を選択
4. Branch を `main`、Folder を `/(root)` にして保存

数分後、表示された Pages URL からプレイできます。

## 主な調整箇所

`game.js` 冒頭付近と各処理内の以下の数値を変更します。

- 重力: `gravity`
- 羽ばたき: `p.vy = Math.min(p.vy - 520, -660)`
- 風の間隔: `random(4.5, 6.5)`
- 風の持続: `updraft.time = 0.72`
- 風の強さ: `4300`
- 攻撃の吹き飛ばし: `attackHitbox()` 内の `kx` / `ky`
