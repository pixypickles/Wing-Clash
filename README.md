# Bird Bowl（1人用 AI対戦版 v4）

`index.html` だけで動作します。GitHub Pagesではリポジトリ直下へ配置してください。

## 操作
- 左右：移動
- 上：ゆるく上昇（羽ばたきより弱い）
- 下：B突進の方向指定
- C／翼：羽ばたき
- Z／A：蹴り上げ
- X／B：突進

### B突進
- 上＋B：短い斜め上突進
- 横または無入力＋B：短い横突進
- 下＋B：短い斜め下突進

スマホでは左下の円形方向キーと、右下の A・B・翼ボタンを使います。黄色がプレイヤー、青色がAIです。

## v4 調整
- 上方向キーを押している間、弱く上昇するようにしました。
- Aボタンを蹴り上げ専用にしました。
- B突進の移動距離と硬直を短くしました。
- ボールの中央への転がり、弱めの重力、巣の下側侵入防止は維持しています。
